# College-Projects

Repository made to storage some college projects. There is a C project, HTML/CSS project and a database project used in Microsoft Access

[1st:](College-Projects/FileSystemSimulator.c) The project that was coded in is a simulator of a file system. Simulating storage space, for files and also managing them, such as creating, deleting, editing
files. 

2nd: The database project was made by me and my group members at the time, we created all the data and meta data for the tables, also adding keys to add relationships
to each table, and lastly, we were asked to make 10 queries to check if the information consistent between the tables. 

3rd: Project was made as a static website, with no backend, just HTML/CSS. It is about financing advice, budgeting, investing, and saving. While also having a page 
to submit a form, about any questions, comments or concerns that you may have. Lastly, a FAQ page.
